Tipos de Listas

Lista Ordenada

Esse tipo de lista segue uma ordenação, e possui tipos de marcadores: 1, A, a, I, i. Segue exemplo.

  1. acordar
  2. tomar cafe
  3. tomar banho
  4. levar trevinha para castrar

Listas não Ordenadas

Como o próprio nome ja diz, trata-se de uma lista não ordenada. Ela possi três types diferentes: disc, circle, square. Segue exemplo de um disc, que seria o padrão de type.

  • acordar
  • tomar cafe
  • tomar banho
  • levar trevinha para castrar

Listas mistas

O possui mesclar os dois tipos de listas citados acima. Segue exemplo:

Supermercado

  • Comidas pereciveis
    1. frango
    2. frutas
    3. legumes
    4. queijo
  • Comidas não pereciveis
    1. arroz
    2. feijão
    3. açucar
    4. molho de tomate
